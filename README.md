🗺️ Página HTML com Image Map (area shape)

Este projeto consiste em uma página HTML que demonstra o uso da funcionalidade <area shape="">, permitindo a criação de áreas interativas dentro de uma imagem.

🚀 Funcionalidades
Uso de image maps com a tag <map> e <area>
Definição de diferentes formatos de área:
rect (retângulo)
circle (círculo)
poly (polígono)
Interatividade ao clicar em regiões específicas da imagem
Redirecionamento ou execução de ações ao clicar nas áreas
🧠 Objetivo

O objetivo deste projeto é praticar e demonstrar como criar zonas clicáveis em imagens utilizando HTML puro, sem necessidade de JavaScript.

🛠️ Tecnologias utilizadas
HTML5
📸 Como funciona

A imagem possui regiões mapeadas onde cada área pode ter:

Links diferentes
Formatos personalizados
Coordenadas específicas
