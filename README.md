<h1>🗺️ Página HTML com Image Map (area shape)</h1>

<p>Este projeto consiste em uma página HTML que demonstra o uso da funcionalidade <area shape="">, permitindo a criação de áreas interativas dentro de uma imagem.</p>

<h2>🚀 Funcionalidades</h2>
<p>
  Uso de image maps com a tag <map> e <area>
  Definição de diferentes formatos de área:
  rect (retângulo)
  circle (círculo)
  poly (polígono)
  Interatividade ao clicar em regiões específicas da imagem
  Redirecionamento ou execução de ações ao clicar nas áreas
  🧠 Objetivo
</p>

<p>
  
  O objetivo deste projeto é praticar e demonstrar como criar zonas clicáveis em imagens utilizando HTML puro, sem necessidade de JavaScript.
</p>
<h2>
  
  🛠️ Tecnologias utilizadas
</h2>
<ul><li>HTML5</li></ul>

<h2>📸 Como funciona</h2>

<p>A imagem possui regiões mapeadas onde cada área pode ter:</p>

<p>
  Links diferentes
  Formatos personalizados
  Coordenadas específicas
</p>
